# 前言

欢迎来到本酒店的管理系统项目分享！本项目是基于Java语言，结合Spring Boot框架、前端技术JS、Vue以及css3，采用MySQL数据库进行开发的实战项目。旨在为毕业生和初学者提供一个实战学习的参考，助你快速掌握Java企业级开发。

# 内容介绍

本项目是一款功能完善的酒店管理系统，主要包括以下模块：客房管理、订单管理、客户管理、员工管理等。系统采用前后端分离的设计，后端采用Java语言和Spring Boot框架，前端采用JS、Vue和css3技术。通过本项目，你可以了解到如何使用Java进行企业级项目的开发，以及如何整合各类技术框架，实现高效、可维护的代码。

# 技术介绍

- 语言：Java
- 使用框架：Spring Boot

# 核心代码

以下是项目中一个简单的Java代码示例，展示了如何使用Spring Boot进行数据库操作：

```java
// 导入Spring Boot相关依赖
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

// 创建一个REST控制器
@RestController
public class RoomController {

    // 注入RoomService
    @Autowired
    private RoomService roomService;

    // 获取所有房间信息
    @GetMapping("/rooms")
    public List<Room> getAllRooms() {
        return roomService.getAllRooms();
    }
}
```
